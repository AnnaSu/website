ABOUT 2013

一直以來，大多數 Web 製作者若是想要學習後端開發，立刻聯想到的語言就是 PHP ； PHP 讓很多人從此踏入 Web 開發的世界且樂此不疲，這是因為 PHP 易學、易用，而且參考資源相當豐富；也因此對許多初學者而言，它是友善且自由的。而從許多國際知名的大型網站 (例如 Yahoo! 、 Facebook ) 是建構在 PHP 之上來看，就足以瞭解 PHP 對於 Web 開發的優勢所在。

然而也因為 PHP 一開始是專注在 Web 開發上，而非朝向真正的語言發展，導致它為此背負了許多舊有的包袱；時至今日，還是讓很多開發者對它抱有很大的誤解。不過近年來， PHP 社群意識到了這些問題，進而提出許多改革方案，使得 PHP 漸漸脫胎換骨，重新出發！

今年， PHPConf 將要為大家帶來新面貌的 PHP ，主題將涵蓋資安、核心、工具、效能、 CMS 模組開發等等。我們將強調以實戰經驗為主，以實例帶領大家親身體驗 PHP 的威力。開發者可以從中學習到業界高手是如何撰寫並善用 PHP ；而企業更可以透過這場聚會，建立起與各方領域高手交流的管道。因此，今年的 PHPConf 是您絕對不能錯過的重要聚會！

SECURITY Allen Own
&
Orange Tsai		 在網路資訊安全越顯重要的今天，您如何保護您的網站資料不被竊取呢？瞭解駭客的心理與技術，瞭解網站程式有何漏洞，都是現在 Web 開發必須熟知的課題。這次， PHPConf 邀請到了 Allen Own 與 Orange Tsai 兩位台灣資安界的高手，他們將透過精心設計的攻防實戰，告訴大家資安的重要性！
TOOLS c9s JavaScript 、 CSS 等這類前端檔案的最小化及相依性管理，常是 Web 開發者頭痛的問題之一。雖然可以借助 Framework 來處理，但也相對地要讓程式碼綁死在 Framework 上面。 PHPConf 邀請到台灣 PHP 界的第一把交椅 c9s ，他將為大家介紹他自己設計的工具： AssetKit ，來更有效率地管理前端檔案。
EXTENSION Rack Lin 效能問題在 PHP 上一直都是很值得探討的議題，而提升執行效能的其中一種方法，就是將關鍵部份以 C 來開發；可是 PHP Extension 的開發對一般開發者來說，門檻相當高；因此 PHPConf 邀請到了技術界人稱阿土伯的 Rack Lin ，他將介紹如何透過 Phalcon Framework 的核心，開發屬於自己的 PHP Extension ！
BIG DATA tsung 當網站規模逐漸擴大時，開發者就一定會面臨到巨量資料的處理。而在台灣，這樣的環境與經驗通常只掌握在大型企業手中，一般開發者很少能接觸到。 PHPConf 邀請到人稱宗董的 tsung ，他將透過開發 SiteTag 的經驗，來為大家說明如何處理巨量資料。
