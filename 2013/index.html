PHP:Reborn

PHPConf Taiwan 2013

10/5

中研院人社館

推文

2012 年，新一代的網路技術大放異彩，而 PHP 也沒有缺席。

不論是 PHP 5.3 與 5.4 版的長足進步，以及各家 PHP Framework 的茁壯，都可以看出 PHP 依舊在 Web 開發這個領域中依舊保持它不可憾動的地位。

而 2013 年，將是 PHP 脫胎換骨的一年。不論在核心、框架、外掛套件等，我們都將會見到一個重生的 PHP ！

期待嗎？ 2013/10/5 來 PHPConf 一探究竟吧！
